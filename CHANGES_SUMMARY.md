# Удаление сердечка из постов

## ✅ Что сделано

Изменения внесены в соответствии с требованием:
> "В Архив можно отправлять новость только из расширенной карточки"

### Изменения:

1. **Удалена кнопка ❤️ из карточек постов в ленте**
   - Файлы: `src/App.tsx`, `src/components/Feed/PostCard.tsx`, `src/components/Feed/VirtualizedFeed.tsx`, `src/components/Feed/index.tsx`
   - Теперь нельзя добавить пост в избранное напрямую из ленты

2. **Оставлена кнопка "📁 В архив" в модальном окне**
   - Файл: `src/App.tsx` (PostDetailModal)
   - Откройте пост → нажмите "В архив" → пост сохранится в разделе "Архив"

3. **Раздел "Архив"**
   - Файл: `src/components/Archive/index.tsx`
   - Здесь отображаются все заархивированные посты

4. **Раздел "Избранное"**
   - Файл: `src/App.tsx` (вкладка favorites)
   - Здесь отображаются **только инструменты/приложения**
   - Посты в избранном больше не отображаются

## 📝 Логика работы

### Для постов:
1. **Просмотр** → Лента новостей
2. **Сохранение** → Открыть пост → "В архив"
3. **Доступ** → Вкладка "Архив"

### Для инструментов:
1. **Просмотр** → Упоминания в посте (синие теги)
2. **Сохранение** → Клик на тег → открыть инструмент → ❤️ сердечко
3. **Доступ** → Вкладка "Избранное"

---

## 🧪 Проверка

1. **Проверьте ленту** - у постов нет сердечка, только кнопка "📄 Подробный саммари"
2. **Откройте пост** - нажмите на карточку
3. **Нажмите "📁 В архив"** - пост должен сохраниться
4. **Перейдите в "Архив"** - пост должен отображаться
5. **Перейдите в "Избранное"** - должны быть только инструменты

## 🗄 База данных

**ВАЖНО:** Для работы архива нужно добавить поле `is_archived` в таблицу `posts`:

```sql
ALTER TABLE posts ADD COLUMN IF NOT EXISTS is_archived BOOLEAN DEFAULT false;
CREATE INDEX IF NOT EXISTS idx_posts_is_archived ON posts(is_archived);
```

👉 Выполнить в: https://supabase.com/dashboard/project/iwtlekdynhfcqgwhocik/sql

Без этого поля кнопка "В архив" не будет сохранять посты в базу!
